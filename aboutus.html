About Me

Hi my name is Morgan, I am a Software Engineer. Here is my GitHub

This assignment was an interesting one for me as I typically spend my time on the back-end of web applications. Being able to put myself in the shoes of the Front-End Engineer that I work with was an eye opening experience as I realized how intricate the work they do is. I left this assignment with a greater understanding of how they perform certain tasks, and will hopefully be able to assist with minor updates if necessary. In the past, I have used tools such as Bootstrap, React and Next.JS in order to quickly create web applications, however, I wanted to challenge myself and see how well I could do creating a website completely from scratch. Overall, this was a nostalgic experience and I particularly enjoyed the Javascript portion of this assignment.

Without a time limit I could spend weeks tweaking this site in order to be satisfied. Ideally I would want to update this website so that it is responsive, and works across all mobile devices. Additionally, given more time, I would also want to create an animation for the home page chart. I think if I added some Javascript libraries I could really make this site look more professional.
